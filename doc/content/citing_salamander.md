# Citing SALAMANDER

This page lists the references to use if you are using SALAMANDER in your publication.
For a list of publications that have used and expanded SALAMANDER, please refer to the [publications.md].

## SALAMANDER

For all publications that use SALAMANDER, please cite the following.

```tex
@misc{SALAMANDER,
    title = {{Software for Advanced Large-scale Analysis of MAgnetic confinement for Numerical Design, Engineering & Research (SALAMANDER). https://github.com/idaholab/salamander}},

    url = {https://github.com/idaholab/salamander},
    institution = {{Idaho National Laboratory and United States Department of Energy, Office of Science (DOE-OS)}}
    year = {2025},
    month = {June}
}
```

## TMAP8

Because SALAMANDER utilizes TMAP8, please refer to [citing_tmap8.md] for all TMAP8-specific citations.

## Cardinal

Because SALAMANDER utilizes Cardinal, please refer to [Citing Cardinal](https://cardinal.cels.anl.gov/publications.html) for all Cardinal-specific citations.

## MOOSE, MOOSE Systems, and MOOSE Physics Modules

Because SALAMANDER is based on MOOSE and several MOOSE-based systems and physics modules,
please refer to [citing.md] for all MOOSE-specific citations.

# SALAMANDER logos

If you want to use the SALAMANDER logo in figures, presentations, website, or other media,
you can find the SALAMANDER logos in `.png` and `.eps` formats in `~/projects/salamander/doc/content/figures`.
These logos include horizontal and vertical alignments, and are available in black, white, and blue.
